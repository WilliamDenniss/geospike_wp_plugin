== Description ==

Adds two Geospike related widgets, feed and travel map.

== Installation ==

1. Upload the geospike folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the admin panel, select Appearance -> Widgets, drag the Geospike Travel Map widget or Geospike Feed widget to the desired section and input your Geospike username.
